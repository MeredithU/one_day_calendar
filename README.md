## One Day Calendar

## This website was built primarily using JavaScript.

## Set Up

1) Clone the repo

2) Open your console and enter: 

	 layOutDay([ {start: 30, end: 150}, {start: 540, end: 600}, {start: 560, end: 620}, {start: 610, end: 670} ]);
